# 四癌篩檢補助資訊網(台中版)

| 淑貞 |   [shujhen][gitgub-shujhen]   | CSS 切版(RWD) + JS 程式（補助資格查詢頁、篩檢機構查詢頁）|
| 若甄 |  [akeymaker][gitgub-akeymaker]  | CSS 切版(RWD) + JS 程式（首頁測驗遊戲、連結區塊）  |
| 智昕 | [petermei82][gitgub-petermei82]  | CSS 切版(RWD) + JS 程式（首頁開頭動畫、四癌資訊、篩檢對象） |
